This document outlines the underlying architecture of our "Living Documentation" platform. By combining static site generation with client-side physics engines, we can create educational materials that react to parameters in real-time.

## 1. The Core Stack

Our platform relies on three distinct layers to deliver interactive content:

* **Starlight (The Framework):** Built on top of Astro, this handles the routing, sidebar generation, typography, and Markdown (MDX) parsing.
* **Astro Components (The Bridge):** `.astro` files act as isolated containers. They allow us to write standard HTML/JS while keeping the CSS scoped globally, preventing style leaks.
* **D3.js (The Physics Engine):** We utilize the `d3-force` module for Verlet integration, particle systems, and continuous animation loops within our components.

---
